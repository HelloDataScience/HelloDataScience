기계학습 모형은 목표변수(Target Variable)의 유무에 따라서 **지도학습(Supervised Learning)**과 **비지도학습(Unsupervised Learning)**으로 나뉩니다. 아래 그림으로 전체 그림을 먼저 살펴보겠습니다.

## 기계학습 알고리즘의 종류

![기계학습 알고리즘 분류]({{ '/assets/images/machine-learning/ml-algorithms-map.png' | relative_url }})

지도학습은 목표변수의 종류에 따라서 **분류(Classification)**와 **추정(Estimation)**으로 나뉩니다. 추정은 **회귀(Regression)**와 혼용해서 언급되기도 합니다. 분류모형은 목표변수가 범주형일 때, 추정(회귀)모형은 목표변수가 연속형일 때 사용됩니다. 입력변수들과 목표변수가 같은 시점이면 추정모형, 목표변수가 입력변수보다 미래시점이면 **예측(Prediction)모형**이 됩니다.

비지도학습은 목표변수가 없다는 특징이 있습니다. **차원축소(Dimension Reduction)**는 p개의 변수를 더 작은 m개로 줄일 때, **군집화(Clustering)**는 전체 데이터를 몇 개의 세부 군집으로 나눌 때, **연관성 규칙(Association Rule)**은 **장바구니 분석(MBA)**처럼 연관성 높은 규칙을 발견할 때 사용합니다.

해결하려는 문제와 데이터의 속성에 따라 어떤 알고리즘을 쓸지 정하는 것이 중요합니다.

## 기계학습에 사용할 데이터셋의 분할

어떤 모형을 적합할 때 전체 데이터를 그대로 쓰는 대신, 다음 두 가지 방법으로 여러 개의 데이터셋으로 나누어 사용합니다.

### 자료분할 (Hold-out Validation)

데이터의 양이 많을 때 주로 사용합니다. 전체 데이터를 **훈련용(training)**, **검증용(validation)**, **시험용(test)** 데이터셋으로 나눕니다. 비중은 분석가가 정할 수 있습니다.

자료분할의 목적은 훈련용 데이터로 만든 모형에 시험용 데이터를 적용해 **과적합 여부**를 판단하는 것입니다. 모형 하나면 훈련용·시험용 둘이면 되지만, 여러 모형 중 최적을 고르려면 별도의 검증용 데이터셋이 필요합니다.

분류모형의 성능은 **혼동행렬(Confusion Matrix)**의 **F1 Score**, **ROC 커브**와 **AUROC**로, 회귀모형은 MSE·RMSE·MAPE 등으로 평가합니다. 공통적으로 실제값과 추정값의 오차(error)를 기준으로 합니다.

### 교차검증 (Cross Validation)

데이터의 양이 적을 때 사용합니다. 먼저 전체를 훈련용·시험용으로 나누고, 훈련용을 k개로 균등하게 다시 나눈 뒤 k-1개로 적합하고 나머지 1개로 검증합니다. 적합에 쓰는 데이터셋을 순차적으로 바꿔가며 k번 반복하므로 **K-겹 교차검증(K-Fold Cross Validation)**이라고도 합니다.

k개의 검증 결과를 평균 내어 모형 성능을 비교하고, 최적 모형이 선택되면 시험용 데이터셋으로 최종 확인합니다.
